Redmine numericfield chart macro

「時系列データ」といえば……

  1. 課員の体温の推移
  2. 勉強会の参加人数の推移
  3. ニコニコカレンダー (気分の推移)
  4. スポーツジムでのトレーニングの進捗
  5. ご家庭の冷蔵庫の野菜の数
  6. お子さんが宿題をやった時間

(このへんでデモを挟むのをわすれないこと)

  • Redmineを起動しておく
  • デモ用データを流し込んでおく
  •  wiki/index を開く
  •  issues を開く

Redmine numericfield chart macro

追加のアイデア

  1. 時間軸を制限したい
    (「過去14日間を表示」など)
  2. ステータス遷移のグラフ
    (Issueの流入や滞留の状況が見える)
  3. 番号じゃなくて検索条件を指定したい
    {{numericfield_chart(CustomField名, クエリ名)}}

Redmine LDRize Plugin とは

  1. j, k キーでカーソルを移動して
  2. pキーでマークしてoキーで開く
  3. 複数を一気に開ける
    (「ポップアップを許可」が必要かも)
  4. my_page / プロジェクトリスト / issue List / issue / wiki index で使える
